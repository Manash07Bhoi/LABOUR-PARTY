# Screens

The application includes the following 17 core screens located in `lib/presentation/screens`:

1.  **Splash Screen:** `lib/presentation/screens/splash/splash_screen.dart`
2.  **Main Shell Screen:** `lib/presentation/screens/shell/main_shell_screen.dart` (Base container/navigation shell)
3.  **Dashboard Screen:** `lib/presentation/screens/dashboard/dashboard_screen.dart`
4.  **Settings Screen:** `lib/presentation/screens/settings/settings_screen.dart`
5.  **About Screen:** `lib/presentation/screens/settings/about_screen.dart`
6.  **Reports Screen:** `lib/presentation/screens/reports/reports_screen.dart`

**Management Screens:**
7.  **Labour Screen:** `lib/presentation/screens/management/labour_screen.dart`
8.  **Add Labour Screen:** `lib/presentation/screens/management/add_labour_screen.dart`
9.  **Driver Screen:** `lib/presentation/screens/management/driver_screen.dart`
10. **Add Driver Screen:** `lib/presentation/screens/management/add_driver_screen.dart`
11. **Tractor Screen:** `lib/presentation/screens/management/tractor_screen.dart`
12. **Add Tractor Screen:** `lib/presentation/screens/management/add_tractor_screen.dart`
13. **Place Screen:** `lib/presentation/screens/management/place_screen.dart`
14. **Add Place Screen:** `lib/presentation/screens/management/add_place_screen.dart`

**Work & Attendance Screens:**
15. **Work Records Screen:** `lib/presentation/screens/work/work_records_screen.dart`
16. **Work Detail Screen:** `lib/presentation/screens/work/work_detail_screen.dart`
17. **Add Work Screen:** `lib/presentation/screens/work/add_work_screen.dart`

*Note: Navigation between these screens is managed via standard Flutter routing or a designated router package.*
